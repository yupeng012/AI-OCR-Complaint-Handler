## 📋 功能详解

### 1. Single & Batch Analysis ✓

**单个分析**:
- 上传图片或输入文本
- 自动 OCR 识别
- 情感分析
- 自动分类
- 生成工单

**批量处理**:
- 支持多图片上传
- 批量文本处理
- 进度条显示
- 批量生成工单
- 批量 Telegram 通知

### 2. Custom Templates ✓

**模板功能**:
- 创建自定义回复模板
- 支持变量：{category}, {ticket_id}, {sentiment}
- 模板保存和加载
- 多模板管理
- 一键应用模板

**示例模板**:
```
Dear Customer,

Thank you for contacting us about {category}.
We take your {sentiment} feedback seriously.

Ticket ID: {ticket_id}

Our team will respond within 24 hours.

Best regards,
Customer Service Team
```

### 3. Excel Export ✓

**导出功能**:
- 一键导出所有工单
- 包含完整信息：
  - Ticket ID
  - Timestamp
  - Category
  - Sentiment
  - Priority
  - Status
  - Text content
- 自动命名：complaints_YYYYMMDD.xlsx
- 支持自定义导出范围

### 4. Real-time Telegram Alerts ✓

**通知功能**:
- 新工单实时推送
- 高优先级告警
- 批量处理通知
- 可配置 Bot Token 和 Chat ID
- 支持开关控制

**通知示例**:
```
🔔 New Complaint
ID: TS2026051112345
Category: Product Quality
Sentiment: Negative
Priority: High
```

## 📊 使用流程

### 单个投诉处理
1. 选择 "Single" 模式
2. 上传图片或输入文本
3. 点击 "Analyze"
4. 查看分析结果
5. 生成工单
6. 生成回复
7. （可选）导出 Excel

### 批量处理
1. 选择 "Batch" 模式
2. 上传多个图片
3. 或输入多行文本
4. 点击 "Process Batch"
5. 查看进度
6. 批量生成工单
7. 导出 Excel

### 自定义模板
1. 侧边栏 → Templates
2. 输入模板名称
3. 编写模板内容（使用变量）
4. 点击 "Save Template"
5. 生成回复时选择模板

### Telegram 配置
在代码中设置：
```python

TELEGRAM_CHAT_ID = "YOUR_CHAT_ID"
```

### Excel 导出
```python
def export_to_excel(tickets):
    df = pd.DataFrame(tickets)
    output = BytesIO()
    with pd.ExcelWriter(output, engine='xlsxwriter') as writer:
        df.to_excel(writer, sheet_name='Tickets', index=False)
    return output.getvalue()
```
