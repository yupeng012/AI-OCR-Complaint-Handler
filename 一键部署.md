## 📦 部署方式

### 方式 1: 本地运行（已启动）
访问：http://localhost:8501 (原服务) 或 http://localhost:8504 (新服务)

### 方式 2: 一键部署到服务器
```bash
# 1. 下载此目录到服务器
# 2. 执行：
./deploy-on-server.sh

# 3. 访问：http://服务器 IP:8501
```

### 方式 3: Docker 部署
```bash
docker build -t ai-complaint .
docker run -p 8501:8501 ai-complaint
```

### 方式 4: Streamlit Cloud
1. 推送到 GitHub
2. 在 streamlit.io/cloud 部署
3. 配置域名 yupeng-ai.cn
