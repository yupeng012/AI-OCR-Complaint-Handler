## 📋 部署步骤（请按顺序执行）

### 步骤 1: 创建 GitHub 仓库（在浏览器中操作）

1. 打开浏览器访问：https://github.com/new
2. 仓库名称：`ai-complaint-handler`
3. 可见性：Public（公开）
4. **不要**初始化 README（我们已经有代码了）
5. 点击 "Create repository"

### 步骤 2: 关联远程仓库并推送

在终端执行以下命令（替换 YOUR_USERNAME 为你的 GitHub 用户名）：

```bash
# 配置 Git 用户（如果之前没有配置过）
git config --global user.name "Yupeng"
git config --global user.email "wtueeq@example.com"

# 添加远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/ai-complaint-handler.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 步骤 3: 部署到 Streamlit Cloud

1. **访问 Streamlit Cloud**: https://streamlit.io/cloud
2. **登录**: 使用 GitHub 账号登录
3. **创建新应用**: 点击 "New App"
4. **选择仓库**: 选择 `ai-complaint-handler`
5. **配置应用**:
   - **Main file path**: `app/unified_app.py`
   - **Python version**: `3.9` (或更高)
   - **Branch**: `main`
6. **点击 "Deploy!"**

### 步骤 4: 配置自定义域名

部署成功后（大约 2-5 分钟）：

1. 在 Streamlit Cloud 仪表板，点击你的应用
2. 点击右上角 "Settings" (设置)
3. 找到 "Custom Domain" 部分
4. 输入：`yupeng-ai.cn`
5. 点击 "Save"
6. 复制显示的 CNAME 记录值（格式类似：`your-app.streamlit.app`）

### 步骤 5: 配置 DNS 记录

在你的域名服务商处（如阿里云、腾讯云、Cloudflare 等）：

**添加 CNAME 记录**:
```
类型：CNAME
主机记录：@ 或 yupeng-ai.cn
记录值：[从 Streamlit 复制的值，如：your-app.streamlit.app]
TTL: 自动或 600
```

**如果使用 Cloudflare**:
1. 登录 Cloudflare 控制台
2. 选择域名 `yupeng-ai.cn`
3. 点击 "DNS" → "Add record"
4. 类型选择 "CNAME"
5. Name: `@`
6. Target: `your-app.streamlit.app`
7. Proxy: 关闭（灰色云朵）
8. 点击 "Save"

### 步骤 6: 等待 DNS 生效

- DNS 生效时间：通常 5-30 分钟，最长 24 小时
- 验证生效：访问 https://yupeng-ai.cn

## ⚠️ 常见问题解决

### 问题 1: Git 推送失败
```bash
# 错误：remote: Repository not found
# 解决：检查仓库名是否正确，确保是 Public 仓库

# 错误：Permission denied
# 解决：检查 GitHub token 或使用 SSH
git remote set-url origin git@github.com:YOUR_USERNAME/ai-complaint-handler.git
```

### 问题 2: Streamlit 部署失败
1. 检查 `requirements_unified.txt` 是否包含所有依赖
2. 查看 Streamlit 部署日志
3. 确保主文件路径是 `app/unified_app.py`

### 问题 3: 域名不生效
```bash
# 检查 DNS 记录
nslookup yupeng-ai.cn

# 清除本地 DNS 缓存
# macOS:
sudo dscacheutil -flushcache
sudo killall -HUP mDNSResponder
# Windows:
ipconfig /flushdns
```

### 问题 4: 导入错误
已在 `app/unified_app.py` 中修复，确保包含：
```python
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent))
```
